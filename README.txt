Mailplane Clean CSS

ABOUT
A CSS file for use with the Mac application Mailplane <http://mailplaneapp.com> by Ruben Bakker of uncomplex gmbh.

Essentially, this simple stylesheet removes the extra clutter in the Gmail interface, making it clean and easy to use. Report any bugs or issues to Jim Mitchell at <http://jimmitchell.org/contact>

USE
To use, save to any location on your Mac, and in Mailplane preferences under the "Tweak UI" option, check "Use custom stylesheet" and then choose the stylesheet where ever you saved it at. Then close the main window in reopen it. The stylesheet apply when the window is refreshed.
